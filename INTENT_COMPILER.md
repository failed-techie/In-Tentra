# INTENTRA Intent Compiler

## 🧠 LLM-Powered Natural Language → Structured Intent

The Intent Compiler uses **LangChain + Groq LLM** to convert natural language trading requests into validated, structured JSON.

---

## 📦 Components

### **1. TradingIntent (Pydantic Model)**

```python
class TradingIntent(BaseModel):
    action: str        # "BUY" or "SELL" (uppercase, validated)
    symbol: str        # Stock ticker (uppercase, e.g., "AAPL")
    amount: float      # Dollar amount in USD (default 100.0)
    confidence: float  # 0.0-1.0 (LLM's confidence in parsing)
    raw_input: str     # Original input text
```

**Validation:**
- `action` must match pattern: `^(BUY|SELL)$`
- `symbol` minimum length: 1
- `amount` must be ≥ 0.01
- `confidence` must be 0.0 ≤ x ≤ 1.0
- Auto-uppercase for `action` and `symbol`

---

### **2. IntentCompiler Class**

```python
class IntentCompiler:
    def __init__(
        model_name: str = "mixtral-8x7b-32768",
        api_key: Optional[str] = None
    )
    
    def compile(raw_input: str) -> TradingIntent
    def compile_batch(raw_inputs: list[str]) -> list[TradingIntent]
    def to_dict(intent: TradingIntent) -> dict
```

---

## 🔧 LangChain Architecture

### **Chain Construction**

```python
# 1. Groq LLM
llm = ChatGroq(
    model="mixtral-8x7b-32768",
    temperature=0.1  # Low temp for consistency
)

# 2. Pydantic Output Parser
parser = PydanticOutputParser(pydantic_object=TradingIntent)

# 3. Prompt Template
prompt = ChatPromptTemplate.from_messages([
    ("system", system_prompt_with_format_instructions),
    ("human", "{raw_input}")
])

# 4. Create chain with LCEL
chain = prompt | llm | parser
```

### **System Prompt**

The system prompt includes:
1. **Role definition** - Expert trading intent parser
2. **Extraction rules** - What to extract and how
3. **Confidence scoring** - Based on input clarity
4. **Edge case handling** - Ambiguity, missing data, unknowns
5. **Examples** - Few-shot learning
6. **Format instructions** - Auto-generated by PydanticOutputParser

## 📝 Example Usage

### **Basic Usage**

```python
from core.intent_compiler import IntentCompiler

# Initialize
compiler = IntentCompiler(
    model_name="mixtral-8x7b-32768",
    api_key="your_groq_api_key"
)

# Parse natural language
intent = compiler.compile("Buy 500 dollars of Tesla stock")

print(intent.action)      # "BUY"
print(intent.symbol)      # "TSLA"
print(intent.amount)      # 500.0
print(intent.confidence)  # ~0.95
```

### **Batch Processing**

```python
inputs = [
    "Buy 500 dollars of Tesla stock",
    "Sell some Apple shares",
    "Maybe buy Microsoft"
]

intents = compiler.compile_batch(inputs)

for intent in intents:
    print(f"{intent.action} {intent.symbol}: ${intent.amount}")
```

### **Convert to Dictionary**

```python
intent = compiler.compile("Sell Apple")
intent_dict = compiler.to_dict(intent)

# {
#   "action": "SELL",
#   "symbol": "AAPL", 
#   "amount": 100.0,
#   "confidence": 0.75,
#   "raw_input": "Sell Apple"
# }
```

## 🎯 Edge Case Handling

### **1. Ambiguous Action**

**Input:** `"Maybe buy something"`

**Output:**
```python
TradingIntent(
    action="BUY",
    symbol="UNKNOWN",
    amount=100.0,
    confidence=0.2  # Low confidence
)
```

### **2. Missing Amount**

**Input:** `"Sell Apple shares"`

**Output:**
```python
TradingIntent(
    action="SELL",
    symbol="AAPL",
    amount=100.0,  # Default
    confidence=0.75
)
```

### **3. Company Name → Ticker**

**Input:** `"Buy $1000 of Microsoft"`

**Output:**
```python
TradingIntent(
    action="BUY",
    symbol="MSFT",  # Converted
    amount=1000.0,
    confidence=0.9
)
```

### **4. Unknown Symbol**

**Input:** `"Buy some XYZ123"`

**Output:**
```python
TradingIntent(
    action="BUY",
    symbol="XYZ123",  # Passed through
    amount=100.0,
    confidence=0.5  # Lower confidence
)
```

### **5. Parse Failure (Fallback)**

If LLM fails completely:

```python
TradingIntent(
    action="BUY",
    symbol="UNKNOWN",
    amount=100.0,
    confidence=0.1,  # Very low
    raw_input=original_input
)
```

## 🎓 Confidence Scoring

| Range | Meaning | Example |
|-------|---------|---------|
| 0.9 - 1.0 | Clear, unambiguous | "Buy 500 dollars of AAPL" |
| 0.7 - 0.8 | Reasonably clear | "Sell some Apple shares" |
| 0.4 - 0.6 | Some ambiguity | "Get me Microsoft stock" |
| 0.1 - 0.3 | Very ambiguous | "Maybe buy something" |

**Use confidence for:**
- Requiring human confirmation (< 0.6)
- Auto-approval thresholds (> 0.8)
- Risk assessment

## 🔒 Error Handling

### **Graceful Degradation**

```python
try:
    intent = compiler.compile(user_input)
    
    if intent.confidence < 0.5:
        # Ask user for confirmation
        confirm_with_user(intent)
        
except Exception as e:
    # Fallback to manual entry
    intent = get_manual_intent()
```

### **Validation Errors**

```python
from pydantic import ValidationError

try:
    intent = TradingIntent(
        action="invalid",  # Will fail pattern validation
        symbol="AAPL",
        amount=500.0
    )
except ValidationError as e:
    print(e.errors())
```

## 🔗 Integration with PolicyEngine

```python
from core.intent_compiler import IntentCompiler
from core.policy_engine import PolicyEngine
from core.enforcement_layer import EnforcementLayer

# Parse natural language
compiler = IntentCompiler()
intent = compiler.compile("Buy 500 dollars of Tesla")

# Convert to policy format
policy_intent = {
    "symbol": intent.symbol,
    "action": intent.action,
    "amount": intent.amount
}

# Validate
policy_engine = PolicyEngine()
enforcement = EnforcementLayer(policy_engine)

result = enforcement.enforce(policy_intent)

if result["decision"] == "ALLOW":
    print("✅ Trade approved!")
else:
    print(f"❌ Trade blocked: {result['reason']}")
```

```
User Input (Natural Language)
    ↓

    ↓

    ↓
ChatGroq LLM (temperature=0.1)
    ↓
PydanticOutputParser
    ↓
TradingIntent (validated)
    ↓
PolicyEngine (deterministic rules)
    ↓
EnforcementLayer (ALLOW/BLOCK)
```

## 💡 Best Practices

1. **Check confidence scores**
   ```python
   if intent.confidence < 0.7:
       # Require human confirmation
   ```

2. **Validate with PolicyEngine**
   ```python
   # Never skip policy validation!
   policy_result = policy_engine.validate(intent_dict)
   ```

3. **Log all intents**
   ```python
   # For audit trail
   logger.info(f"Parsed: {intent.raw_input} → {intent.model_dump()}")
   ```

4. **Handle fallbacks**
   ```python
   if intent.symbol == "UNKNOWN":
       # Ask user to clarify
   ```

5. **Use batch for efficiency**
   ```python
   # Process multiple intents at once
   intents = compiler.compile_batch(user_inputs)
   ```
